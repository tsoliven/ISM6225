Experience Developing This Website

Working on this assignment provided hands-on experience in designing a website using HTML, CSS, and JavaScript. My assigned theme was "Luxury Hotel", so I utilized a dark blue and gold color palatte to give off an expensive and luxorious feel.

The process included building the page structure and content on HTML and adding styles and layouts using CSS. The mock reservation utilizes JavaScript and demonstrates a simplified CRUD (Create, Read, Update, Delete) simulation for interactivity.

Files were published via Cyberduck (SFTP) to public_html on myweb.usf.edu and verified for cross-browser compatibility.

Technologies Used

  • HTML5: Semantic structure and accessibility
  • CSS3: Custom variables, grid/flex layouts, responsive design
  • JavaScript (ES6): Page interactivity and CRUD simulation
  • Chart.js: Interactive charts for analytics visualization
  • Cyberduck: SFTP upload for live hosting
